## ?? Quick Start
1. `npm ci`
2. Copy `.env.example` to `.env` and configure your environment
3. `node scripts/migrate-secrets.js --provider=aws` (or `--provider=gcp`)
4. `npm run dev`

## ?? Deployment Options
